# Manual Test Plan for Hero Text Animation

## Objective
Ensure the hero text animation on the CV/portfolio website appears modern and polished without compromising accessibility or performance.

## Tests

### Compatibility
- [ ] Verify the animation works on the latest versions of Chrome, Firefox, Safari, and Edge.
- [ ] Test responsiveness across different screen sizes, including mobile and tablet.

### Performance
- [ ] Ensure the animation loads without noticeable delay on both desktop and mobile.

### Accessibility
- [ ] Check that prefers-reduced-motion settings disable animation for users who have it enabled.
- [ ] Confirm text is readable and maintains sufficient contrast throughout the animation.

### User Experience
- [ ] Verify the animation does not loop endlessly and settles into a stable state.
- [ ] Ensure the animation enhances rather than detracts from the page's aesthetic appeal.
